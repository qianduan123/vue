# vue 后端项目基础模板

> A Vue.js project

## Build Setup

``` bash
> 开发 npm run dev

> 输出静态文件夹 npm run build
```

```
├── README.md
├── build                       // 构建
├── config                      // 构建参数
├── package.json
├── src
│   ├── App.vue                 // 根组件
│   ├── assets                  // 资源文件  
│   ├── components              // 公共组件
│   ├── config                  // 配置文件
│   ├── filters                 // 暂无
│   ├── main.js                 // 根文件
│   ├── router                  // 路由
│   ├── server                  // 接口服务
│   ├── store                   // vuex
│   │   ├── index.js
│   │   └── modules
│   ├── util                    // 工具
│   └── views                   // 页面
│       ├── account

```
